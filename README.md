## Install

__```NOTE :```__ <code>__```Ensure you have Node.JS version 4+ installed. Then run the following:```__</code>

```
$ npm install --global xkcd-comics
```

__OR__

```
$ sudo npm install --global xkcd-comics
```

## Usage

```
$ xkcd
```
