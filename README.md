## Install

```
$ npm install --global xkcd-comics
```

__OR__

```
$ sudo npm install --global xkcd-comics
```

## Usage

```
$ xkcd
```

__`NOTE : `__ `Strips will be saved under` __`xkcd-strips folder`__ `in your` __`home directory`__
